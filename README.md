# Like Me — Parte I

Aplicación fullstack para gestión de posts con likes.

## Stack
- React + Vite
- Node + Express
- PostgreSQL

## Instalación

### Backend
cd server  
npm install  
npm run dev  

### Frontend
cd client  
npm install  
npm run dev  

## Base de datos
Crear DB `likeme` con tabla:

- id
- titulo
- img
- descripcion
- likes
